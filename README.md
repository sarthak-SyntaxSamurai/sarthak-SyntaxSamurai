### 🐸 Featured Project: **[FrogDrop](https://github.com/sarthak-SyntaxSamurai/FrogDrop)**

A tiny, feature-packed companion that lives in your macOS Menu Bar:
* 📦 **Dropzone** — Drag files to your menu bar to shelf or batch-move them instantly.
* 📋 **Clipboard Manager** — Secure local clipboard history with custom rules and TinyURL shortener.
* ⏱️ **Focus Timer** — Pomodoro/Stopwatch with an interactive blooming flower progress dashboard.
* 🍻 **Homebrew Cask** — Bypasses macOS Gatekeeper warnings natively during install.

**Quick Install:**
```bash
brew tap sarthak-SyntaxSamurai/tap && brew install --cask frogdrop
```
